====================

Run your nodejs service in style (think php-fpm for nodejs)

**Version Matrix:**

node version | node-pm version
------------ | ---------------
\>= 0.10.0 | >= 0.11.0
0.8.x | <= 0.10.6

Installation
====================

### Install node-pm

``` bash
    npm install -g node-pm
```

Usage
====================

### Run application

``` bash
    node-pm app.js
```

### List node-pm options

``` bash
    node-pm
```

Run Tests
====================

``` bash
    npm test
```

Generate Docs
====================

### Install jsdoc

``` bash
    npm install -g jsdoc
```

### Generate Docs

``` bash
    npm run-script docs
```
